# Recipes App

A SPA application for displaying a list of posts and detailed information about each post.
The data is fetched from the DummyJSON API.

## Features
- React 19 with hooks
- TanStack Router 1.158.0 for routing
- TanStack Query 5.90.20 for work with REST API
- Redux Toolkit for state management
- Vite 7.2.4 for building the project
- Responsive design for mobile and desktop
- WebSocket Chat Module (wss://ws.ifelse.io)

## Pages
- Home '/',
- Recipe '/post/$id'

## Technical Features
- Fetch posts list from API
- View post details
- Interactive with likes/dislikes and views for every post
- Filtration posts by tags
- Searching posts by name
- Pagination
- Support Chat
